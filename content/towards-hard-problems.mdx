# Why pick hard problems to solve?

Over the past few months, I've seen many AI startups creating agents. So far, I'd like to say that there are three disctint type so of AI agents I'm seeing. First are creator agents, whcih create software and products to let the end user of the created software and product create other things. Rplit, Cursor, Bolt, etc. These incldue all of the coding assistatns as well. 

The second type are all around sales and marketing. These include contnent generation, automated marketing, sales pipeline, etc. LLMs are amazing at creating things, after all, their most basic use case is to guess the next best word.

And then, third are the others. The AI agents trying to solve hard problems. 

On a side note, I dont' want to put chatbots as agents. There are qutie a few specalized chatbots for specific purposes, but I don't quite think a chatbot qualifies as an agent.
